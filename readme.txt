A little slider that invites users to share the post.

== Description ==

Active Share is a free plugin provided by [OrangeSoda](http://orangesoda.com) and written by Dan Garfield. For marketing tips [follow OrangeSoda on twitter](http://twitter.com/orangesoda). This plugin was inspired by a similar feature on okcupid's blog.

== Installation ==
1. Upload the `active-share` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where can I change settings? =

Under "Settings" there is a menu item called "Active Share Settings"
